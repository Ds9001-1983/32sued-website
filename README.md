# 32Süd – Restaurant, Café & Bar | Gummersbach

Offizielle Website des Restaurant 32Süd auf dem historischen Steinmüllergelände in Gummersbach. Moderne bergische Küche in industriellem Ambiente.

## Tech Stack

- **Framework:** Next.js 16 (App Router)
- **Sprache:** TypeScript
- **Styling:** Tailwind CSS 4
- **Animationen:** Framer Motion
- **Icons:** Lucide React
- **Formulare:** React Hook Form
- **Deployment:** Vercel-ready

## Seiten

| Seite | Route | Beschreibung |
|-------|-------|-------------|
| Startseite | `/` | Hero, Öffnungszeiten, Story-Teaser, Wochenkarte, Catering-Teaser, Kontakt |
| Restaurant | `/restaurant` | Geschichte, Philosophie-Cards, Galerie mit Lightbox |
| Speisekarte | `/speisekarte` | Mittags- und Abendkarte mit Tab-Navigation |
| Catering | `/catering` | Event-Service, Leistungen, Galerie, Anfrageformular |
| Kontakt | `/kontakt` | Kontaktdaten, Google Maps, Kontaktformular |
| Impressum | `/impressum` | Rechtliche Angaben gemäß § 5 TMG |
| Datenschutz | `/datenschutz` | DSGVO-konforme Datenschutzerklärung |

## Features

- Responsive Design (Mobile-First)
- DSGVO-konformer Cookie-Banner
- SEO-optimiert mit JSON-LD Structured Data
- Framer Motion Scroll-Animationen
- Parallax-Effekte
- Lightbox-Galerie
- Sticky Navigation mit Scroll-Effekt
- Kontakt- und Event-Anfrageformulare

## Entwicklung

```bash
npm install
npm run dev
```

## Deployment

Das Projekt ist Vercel-ready. Einfach das Repository mit Vercel verbinden.

```bash
npm run build
```
